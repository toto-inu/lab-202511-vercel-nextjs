vercelとneonのcliを事前にインストール & 認証をしておく

## Vercel
npm install -g vercel
vercel login

--- コマンドリスト ---
vercel --version
vercel whoami 


## Neon
npm install -g neonctl
neonctl auth

--- コマンドリスト ---
neonctl projects list
neonctl databases list --project-id=[プロジェクトID]
neonctl branches create --project-id=[プロジェクトID]
neonctl connection-string [branch名]
